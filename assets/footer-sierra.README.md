# Imagen del footer — sierra

El footer usa como fondo (muy tenue, sobre negro oliva) la ilustración de la
sierra. Sube el archivo aquí con este nombre exacto:

    assets/footer-sierra.jpg

- Vale la **última foto** que pasaste (el dibujo de Sierra Mágina con el logo AURO).
- JPG horizontal, ~1600–2400 px de ancho.
- Se muestra con opacidad baja (~16%) y un degradado oscuro encima, así que el
  texto del footer siempre se lee. No hace falta que sea oscura.
- Si el archivo no está, el footer simplemente queda en negro oliva liso (no rompe nada).
