# Fotos del proceso de obtención

Suelta aquí 6 fotos con estos nombres exactos. Aparecen automáticamente en el
carrusel deslizable "El proceso" del index (formato tipo *stories* de Instagram,
pensado para móvil). Las tarjetas son **verticales (4:5)**, así que van perfectas
fotos en vertical o cuadradas; una horizontal también sirve (se recorta al centro).

| Archivo  | Qué foto poner |
|----------|----------------|
| `01.jpg` | **Recolección**: aceituna verde en la rama / recogida en el olivar |
| `02.jpg` | **Al molino**: aceituna recién cogida (cajas/remolque) o lavado |
| `03.jpg` | **Molturación**: la pasta de aceituna / el molino |
| `04.jpg` | **Batido en frío**: batidora/termobatidora, pasta removiéndose |
| `05.jpg` | **Extracción**: centrífuga / decánter, el aceite saliendo |
| `06.jpg` | **Envasado**: el aceite dorado / llenado de la lata |

Consejos:
- **JPG**, ~200–400 KB cada una (1200–1600 px de ancho va sobrado para web).
- Mismas proporciones (ideal 4:5, vertical) para que el carrusel quede uniforme.
- Si una falta, se muestra un degradado de marca con el nombre del paso (sin foto).
- ¿Las tienes con otros nombres? Súbelas igual y dime, las renombro yo
  (como hicimos con la lata).
